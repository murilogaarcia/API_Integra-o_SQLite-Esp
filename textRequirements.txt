Simulando sensores (simulador.py)
import requests
import random
import time

URL = "http://127.0.0.1:8000/dados/"

while True:
    sensor_id = random.randint(1, 3)
    velocidade = round(random.uniform(30, 130), 2)

    response = requests.post(URL, params={
        "sensor_id": sensor_id,
        "velocidade": velocidade
    })

    print(f"Sensor {sensor_id} registrou {velocidade} km/h → Resposta da API:", response.json())
    time.sleep(2)

Executando o simulador

Em outro terminal mantendo a API rodando:

python simulador.py
    -envia dados aleatórios a cada 2 segundos.
    -no terminal respostas da API indicando que o dado foi gravado.
    -dados gravados vão para velocidades.db.
